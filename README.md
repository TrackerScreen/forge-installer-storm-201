## Installation

```bash
git clone https://github.com/TrackerScreen/forge-installer-storm-201.git
cd forge-installer-storm-201
```

## Usage

```bash
python setup.py
```

`setup.py` will:

1. Download the official installer (`.exe` or `.msi`) from a direct URL.
2. Verify the SHA-256 hash (if provided).
3. Run a silent install.
4. Launch the application.
